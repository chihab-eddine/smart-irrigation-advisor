Cahier des Charges
Projet : Smart Irrigation & Crop Disease Detection System for Moroccan Agriculture
________________


1. Contexte du Projet
L’agriculture représente un secteur stratégique au Morocco, mais elle fait face à plusieurs défis majeurs :
* stress hydrique et sécheresse
* gaspillage d’eau dans l’irrigation
* difficulté de détection précoce des maladies des cultures
* manque d’outils numériques accessibles aux petits agriculteurs
Dans ce contexte, l’utilisation de l’intelligence artificielle et des données météorologiques peut permettre :
* d’optimiser l’utilisation de l’eau
* de réduire les pertes agricoles
* d’améliorer le rendement des cultures
Le projet consiste à développer une plateforme intelligente permettant :
1. de recommander des stratégies d’irrigation adaptées
2. de détecter automatiquement certaines maladies des plantes à partir d’images
________________


2. Problématique
Les agriculteurs disposent rarement :
* d’outils prédictifs simples
* d’analyses météorologiques exploitables
* de systèmes automatiques de diagnostic des maladies
Les décisions sont souvent prises manuellement et de manière approximative, ce qui entraîne :
* une surconsommation d’eau
* des pertes de rendement
* une propagation rapide des maladies
La problématique principale est donc :
Comment utiliser l’intelligence artificielle et les données météorologiques pour assister les agriculteurs marocains dans l’optimisation de l’irrigation et la détection précoce des maladies des cultures ?
________________

4. Description Générale de la Solution
La plateforme sera composée de deux modules principaux :
________________


4.1 Module Smart Irrigation
L’utilisateur saisit :
* le type de culture
* la région
* le type de sol
Le système :
* récupère les données météorologiques
* analyse les paramètres climatiques
* estime les besoins hydriques
* affiche des recommandations d’irrigation
________________


4.2 Module Disease Detection
L’utilisateur :
* importe une image d’une feuille de plante
Le système :
* analyse l’image via un modèle CNN
* détecte la maladie éventuelle
* retourne :
   * le nom de la maladie
   * le pourcentage de confiance
   * des conseils de traitement
________________
